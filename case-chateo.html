Chateo 🔒

Secure End-to-End Encrypted Messaging

Project Capstone Project
Year 2025-2026
Platform Android & iOS
🚀 Coming Soon to Play Store
01

App Preview

Chateo is a secure messaging application I developed as my capstone project. The app features military-grade AES-256 encryption, ensuring that all messages remain private and secure between users.

Built with Flutter for cross-platform compatibility, Chateo combines modern UI design with robust security features, making secure communication accessible to everyone on both Android and iOS.

AES-256 Encryption
2 Platforms
Real-time Messaging
02

The Challenge

Privacy Concerns

Many messaging apps lack proper end-to-end encryption or have complex privacy policies.

Security vs UX

Balancing strong security features with an intuitive, user-friendly interface.

Cross-Platform

Building a single codebase that works seamlessly on both Android and iOS devices.

Real-time Performance

Ensuring instant message delivery while maintaining encryption overhead.

03

The Solution

01

AES-256 End-to-End Encryption

Military-grade encryption ensures all messages are completely secure. Only the sender and recipient can read the messages — not even the server can decrypt them.

02

Real-Time Messaging with Socket.IO

Instant message delivery with read receipts and typing indicators. Messages appear in real-time without any noticeable delay.

03

Firebase Authentication

Secure user authentication with Firebase ensures only authorized users can access the app. Supports email/password and social login options.

04

Flutter Cross-Platform Development

Single codebase powers both Android and iOS apps with native performance. Material Design ensures a modern, intuitive user interface.
